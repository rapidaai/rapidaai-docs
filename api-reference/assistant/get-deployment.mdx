---
title: "Get Deployment"
description: "Get the latest assistant deployment from the Rapida SDK."
---

Gets the latest deployment for an assistant. Use the deployment-specific SDK function for the deployment channel you want to retrieve.

<Info>
  For setup and authentication options, see the [Installation guide](/api-reference/installation) and [Authentication](/api-reference/authentication).
</Info>

## Deployment functions

| Deployment | React / Node.js | Python client method | Go |
| --- | --- | --- | --- |
| API | `GetAssistantApiDeployment` | `GetAssistantApiDeployment` | `clients.GetAssistantApiDeployment` |
| Debugger | `GetAssistantDebuggerDeployment` | `GetAssistantDebuggerDeployment` | `clients.GetAssistantDebuggerDeployment` |
| Phone | `GetAssistantPhoneDeployment` | `GetAssistantPhoneDeployment` | `clients.GetAssistantPhoneDeployment` |
| Web plugin | `GetAssistantWebpluginDeployment` | `GetAssistantWebpluginDeployment` | `clients.GetAssistantWebpluginDeployment` |
| WhatsApp | `GetAssistantWhatsappDeployment` | `GetAssistantWhatsappDeployment` | `clients.GetAssistantWhatsappDeployment` |

## Parameters

<ParamField body="connectionConfig" type="ConnectionConfig" required>
  Configuration for the client connection.
</ParamField>

<ParamField body="request" type="GetAssistantDeploymentRequest" required>
  Deployment lookup request.
  <Expandable>
    <ParamField body="assistantId" type="uint64" required>
      Assistant ID for the deployment.

  </Expandable>
</ParamField>

```ts React
import {
  ConnectionConfig,
  GetAssistantApiDeployment,
  GetAssistantDeploymentRequest,
} from "@rapidaai/react";

const auth = ConnectionConfig.WithDebugger({
```

```python Python
import os
from pprint import pprint

from rapida import ConnectionConfig
from rapida.clients.protos.assistant_deployment_pb2 import (
    GetAssistantDeploymentRequest,
)

auth = ConnectionConfig.with_debugger(
    authorization=os.getenv("RAPIDA_AUTHORIZATION"),
    user_id=os.getenv("RAPIDA_AUTH_ID"),
    project_id=os.getenv("RAPIDA_PROJECT_ID"),
)
connection_config = ConnectionConfig.default_connection_config(auth)

response = connection_config.assistant_deployment_client.GetAssistantApiDeployment(
    GetAssistantDeploymentRequest(
        assistantId=int(os.getenv("RAPIDA_ASSISTANT_ID")),
    ),
    metadata=connection_config.auth,
)
pprint(response)
```

```go Go
package main

import (
	"context"
	"fmt"
	"os"

	clients "github.com/rapidaai/rapida-go/rapida/clients"
	connections "github.com/rapidaai/rapida-go/rapida/connections"
	protos "github.com/rapidaai/rapida-go/protos"
)

func main() {
	connectionConfig := connections.DefaultconnectionConfig(
		connections.WithPersonalToken(
			os.Getenv("RAPIDA_AUTHORIZATION"),
			os.Getenv("RAPIDA_AUTH_ID"),
			os.Getenv("RAPIDA_PROJECT_ID"),
		),
	)

	response, err := clients.GetAssistantApiDeployment(
		context.Background(),
		connectionConfig,
		&protos.GetAssistantDeploymentRequest{
			AssistantId: 2227823180112723968,
		},
	)
	if err != nil {
		fmt.Printf("Error while getting deployment: %+v\n", err)
		return
	}

	fmt.Printf("Deployment response: %+v\n", response)
}
```
