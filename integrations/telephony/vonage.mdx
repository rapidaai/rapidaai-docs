## Getting Started

To integrate Vonage with your Rapida application, follow these steps:

## Prerequisites

- An active Vonage account
- Access to Vonage's API credentials

# Setting Up Provider Credentials

<Steps>
<Step title="Access the Integrations Page">
Navigate to the "Integration > Tools" page. Here you'll see a grid of various external integrations including Vonage.
![View Provider Credential](/images/integration/external/external.png)

</Step>

<Step title="Select Vonage">
On the Integrations page, find the Vonage card. Click the "Setup Credential" button for Vonage.

</Step>

<Step title="Create Provider Credential">
![View Provider Credential](/images/integration/external/create-provider-credential.png)
A modal window will appear titled "Create provider credential". Follow these steps:

1. Ensure Vonage is selected from the dropdown
2. Enter a Key Name: Assign a unique name to this Vonage credential for easy identification
3. Enter the required credentials:
   - Vonage API Key
   - Vonage API Secret
   - Vonage Application ID (if applicable)
4. Click "Configure" to save the credentials

</Step>

<Step title="Verify Credential Setup">
After setting up the credential, you can verify it's been added:

1. The Vonage card should now show "Connected"
2. If you click on Vonage, you'll see a "View provider credential" modal
3. This modal displays the credential name, when it was last updated, and options to delete or close

</Step>
</Steps>

Your Vonage credential is now set up and ready to use with the integration system.

## Features

Vonage integration provides the following features:

- Voice Calls: Make outbound calls and receive inbound calls
- SMS: Send and receive text messages
- Video: Implement video calling and conferencing
- Number Insight: Get detailed information about phone numbers
- Verify: Implement two-factor authentication (2FA)

## Usage Examples

Here are some common use cases for Vonage integration:

1. Customer Support: Set up a cloud-based call center for your support team
2. Appointment Reminders: Send automated voice or SMS reminders to customers
3. Two-Factor Authentication: Use SMS or voice calls for secure user verification
4. Lead Management: Automate outbound calls for sales and marketing campaigns
5. Video Conferencing: Implement secure video meetings for remote teams

For more information on Vonage's API and capabilities, visit the [Vonage API Developer Documentation](https://developer.vonage.com/).
