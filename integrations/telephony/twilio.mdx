## Getting Started

To integrate Twilio with your Rapida application, follow these steps:

## Prerequisites

- An active Twilio account
- Access to Twilio's API credentials

# Setting Up Provider Credentials

<Steps>
<Step title="Access the Integrations Page">
Navigate to the "Integration > Tools" page. Here you'll see a grid of various external integrations including Twilio.
![View Provider Credential](/images/integration/external/external.png)

</Step>

<Step title="Select Twilio">
On the Integrations page, find the Twilio card. Click the "Setup Credential" button for Twilio.

</Step>

<Step title="Create Provider Credential">
![View Provider Credential](/images/integration/external/twilio_credentials.png)
A modal window will appear titled "Create provider credential". Follow these steps:

1. Ensure Twilio is selected from the dropdown
2. Enter a Key Name: Assign a unique name to this Twilio credential for easy identification
3. Enter the required credentials:
   - Twilio Account SID
   - Twilio Auth Token
4. Click "Configure" to save the credentials

</Step>

<Step title="Verify Credential Setup">
After setting up the credential, you can verify it's been added:

1. The Twilio card should now show "Connected"
2. If you click on Twilio, you'll see a "View provider credential" modal
3. This modal displays the credential name, when it was last updated, and options to delete or close

</Step>
</Steps>

Your Twilio credential is now set up and ready to use with the integration system.

## Features

Twilio integration provides the following features:

- Voice Calls: Make outbound calls and receive inbound calls
- SMS and MMS: Send and receive text and multimedia messages
- Video: Implement video conferencing capabilities
- WhatsApp: Send and receive WhatsApp messages
- Programmable Voice: Create custom voice applications
- Two-Factor Authentication: Implement secure user verification

## Usage Examples

Here are some common use cases for Twilio integration:

1. Customer Support: Set up a cloud-based contact center for your support team
2. Appointment Reminders: Send automated voice or SMS reminders to customers
3. Two-Factor Authentication: Use SMS or voice calls for secure user verification
4. Lead Management: Automate outbound calls and messages for sales and marketing campaigns
5. Survey Collection: Conduct voice or SMS-based surveys for customer feedback

For more information on Twilio's API and capabilities, visit the [Twilio Developer Documentation](https://www.twilio.com/docs).
