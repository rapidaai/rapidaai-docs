## Getting Started

To integrate Exotel with your Rapida application, follow these steps:

## Prerequisites

- An active Exotel account
- Access to Exotel's API credentials

# Setting Up Provider Credentials

<Steps>
<Step title="Access the Integrations Page">
Navigate to the "Integration > Tools" page. Here you'll see a grid of various external integrations including Exotel.
![View Provider Credential](/images/integration/external/external.png)

</Step>

<Step title="Select Exotel">
On the Integrations page, find the Exotel card. Click the "Setup Credential" button for Exotel.

</Step>

<Step title="Create Provider Credential">
![View Provider Credential](/images/integration/external/exotel.png)
A modal window will appear titled "Create provider credential". Follow these steps:

1. Ensure Exotel is selected from the dropdown
2. Enter a Key Name: Assign a unique name to this Exotel credential for easy identification
3. Enter the required credentials:
   - Exotel API Key
   - Exotel API Token
   - Exotel SID (Account SID)
4. Click "Configure" to save the credentials

</Step>

<Step title="Verify Credential Setup">
After setting up the credential, you can verify it's been added:

1. The Exotel card should now show "Connected"
2. If you click on Exotel, you'll see a "View provider credential" modal
3. This modal displays the credential name, when it was last updated, and options to delete or close

</Step>
</Steps>

Your Exotel credential is now set up and ready to use with the integration system.

## Features

Exotel integration provides the following features:

- Voice Calls: Make outbound calls and receive inbound calls
- SMS: Send and receive text messages
- Number Masking: Protect customer and agent phone numbers
- Call Recording: Record and store call conversations
- IVR (Interactive Voice Response): Set up automated phone systems

## Usage Examples

Here are some common use cases for Exotel integration:

1. Customer Support: Set up a cloud-based call center for your support team
2. Appointment Reminders: Send automated voice or SMS reminders to customers
3. Two-Factor Authentication: Use SMS or voice calls for secure user verification
4. Lead Management: Automate outbound calls for sales and marketing campaigns
5. Survey Collection: Conduct voice-based surveys for customer feedback

For more information on Exotel's API and capabilities, visit the [Exotel Developer Documentation](https://developer.exotel.com/).
